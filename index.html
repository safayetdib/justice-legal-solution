Flexbox is a one-dimensional layout module in CSS which is used to hold items in a single row or column. To create flexible and responsive navigation menus, form elements and other small scale layouts flexbox is useful.
On the other hand, Grid is a two-dimensional layout module in CSS, which means in can create complex grid layout in both x and y axis. It is helpful for creating flexible and responsive rows and columns in large scale projects.

Media Queries allows a developer to create responsive designs. It starts with the @media keyword and specifies a set of CSS styles that should be applied only when certain conditions are met depending on the screen size, resolution and type of device.

To determine the size and position of elements on a web page web browsers use a layout model called CSS Box Model. The box model consists of the following parts:
-> Content (the actual content of the element)
-> Padding (the area between the content and the border)
-> Border (the area around the padding)
-> Margin (the area outside the border)

Semantic tags are HTML tags that are used to provide additional context and meaning to the content on a web page, making it more accessible and easier to understand for both users and search engines. Examples of semantic tags include:
-> header
-> nav
-> article
-> section
-> aside
-> footer etc.
